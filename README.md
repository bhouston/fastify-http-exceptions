## fastify-http-exceptions monorepo

This repo contains the `fastify-http-exceptions` package and a small demo app.

- `packages/fastify-http-exceptions`: core library and Fastify plugin.
- `demos/basic-example`: demo Fastify app using the plugin.

Use `pnpm tsc`, `pnpm check`, and `pnpm test` at the root to work with all packages.
